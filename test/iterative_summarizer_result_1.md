The Transformer is a new network architecture based solely on attention mechanisms, dispensing with recurrence and convolutions entirely. It achieves superior quality in machine translation tasks, demonstrating better parallelization and significantly less training time compared to dominant recurrent or convolutional models. On the WMT 2014 English-to-German translation task, the big Transformer model achieved a new state-of-the-art BLEU score of 28.4, outperforming previous models by more than 2.0 BLEU. Unlike prior convolutional models where operations to relate distant positions grow with distance, the Transformer reduces this to a constant number of operations. Self-attention layers connect all positions with a constant number of sequentially executed operations, which results in shorter path lengths between any two positions. This constant path length significantly aids in learning long-range dependencies, a key challenge in sequence transduction tasks. Its architecture features an encoder-decoder structure, utilizing stacked self-attention and point-wise fully connected layers for both components. To account for sequence order, the model injects positional encodings, added to input embeddings, which use sine and cosine functions of varying frequencies. This allows the model to extrapolate to sequence lengths longer than those encountered during training. The attention function maps a query and key-value pairs to an output, computed as a weighted sum of values. This mechanism is implemented using Scaled Dot-Product Attention, which computes weights by scaling dot products, and Multi-Head Attention, which allows the model to jointly attend to different representation subspaces. Training involved the Adam optimizer and regularization techniques like Residual Dropout and Label Smoothing. The Transformer also demonstrated strong performance on English constituency parsing, outperforming previous models even with limited training data. Experiments revealed that reducing the attention key size (`dk`) negatively impacts quality, while learned positional embeddings yielded results nearly identical to the sinusoidal ones.
